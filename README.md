# 🚀 NextPlan-AI

Adaptive Weekly Intelligence Planner

## 🧠 About

NextPlan-AI is a dynamic weekly scheduling system that:

- Generates full 7-day intelligent plans
- Uses 30-minute internal precision
- Merges tasks into natural readable blocks
- Tracks completion with persistent checkboxes
- Provides AI-based daily suggestions
- Distributes workload dynamically
- Automatically keeps weekends lighter

## 🏗 Architecture

- Weekly Grid Engine
- Task Allocation Engine
- Merge & Display Engine
- Suggestion Intelligence Panel
- Persistent History Tracking

## 📦 Tech Stack

- Python
- Streamlit

```bash
pip install streamlit
streamlit run app.py
